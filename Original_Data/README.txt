README

All datasets were donwloaded from Open Science Framework Repository. From all original datasets, 
no observation was removed, but only relevant columns were kept and renamed for clarity.
ID was kept to allow further analysis. 

The MOCA Dataset has been downloaded from https://osf.io/y4jzs/
The GEMS Dataset has been downloaded from https://osf.io/4t5a8/
The Tele-GEMS Dataset has been downloaded from https://osf.io/r3ta5/
